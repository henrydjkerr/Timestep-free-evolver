Parameters for CUDA-implemented firing time finder

Technical
=========
version:IMPORT_WAVE
|Number of threads per block
threads:256

|Target number of spikes sought
spikes_sought:1500

|How close to the first firing time you need to be in relative terms to qualify as simultaneous firing
|Should be non-negative
leniency_threshold:0.00001

|Margin of agreement for successive Newton approximations to be considered a final result
|Should be positive
error_bound:0.0000001


Dimensions
==========
|Number of neurons in each grid dimension
|If the number is 1 or less, that dimension will be neglected
neuron_count_x:2000
neuron_count_y:1
neuron_count_z:1

|Spacing between neurons in each dimension
|Should be non-negative
dx:0.01
dy:0.1
dz:1

|Offsets for even rows, to create hexagonal patterns
|offset_yx will move every even y row in the x direction
|offset_zx will move every even z layer in the x direction
|offset_zy will move every even z layer in the y direction
|Should be non-negative, will multiply by -1 if not
even_offset_yx:0
even_offset_zx:0
even_offset_zy:0

Note- you are obliged to use the x, y, z axes as your first, second and third dimensions.  The program reads neuron_count_x, y, z in sequence and ignores any further values when it finds the first value <= 1.




Equation parameters
===================
v_r:0
|Should be larger than v_r
v_th:1

|Don't set this to 1 (or almost 1) for now or else you'll get divide-by-zero errors
|Should be non-negative
synapse_decay:4
|How small (synapse_decay - 1) is allowed to get before you swap to 
|special equations
decay_tolerance: 0.00001

#Difference of normal curves

exp_strength_e:11

Input strength options
======================
|Determines gaussian curve spread for input strength curve proxy
|Should be non-negative
input_sigma:1
input_strength:20

|Determines time the input "shuts off" at, if such an option is used
input_shutoff:3

|Baseline input values before and after shutoff
input_base_before:1
input_base_after:1

|Modifiers for cosh-reciprocal curve shape
rc_height:2
rc_narrow:10
